Команды Git

  • COMMIT - команда используется для сохранения изменений, сделанных в локальном репозитории, в истории коммитов.
  • BRANCH - команда позволяет создавать, просматривать, переименовывать и удалять ветки в Git.
  • CHECKOUT - команда, которая позволяет переключаться между ветками, перемещаться по коммитам, восстанавливать файлы из предыдущих состояний и не только.
  • RESET - команда, позволяет сбросить историю коммитов до определённого момента.
  • REBASE - команда, позволяющий объединить изменения двух веток.
  • MERGE - команда, которая выполняет слияние веток, то есть добавляет изменения из одной ветки в другую.
  • CHERRY-PICK - команда, позволяет выбрать конкретные коммиты из одной ветви и применить их к другой ветви.
